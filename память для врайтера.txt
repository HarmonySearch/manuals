WIN позволяет установить ограничение памяти для soffice.bin. Это добавляет раздел Win32 и два ключа в bootstrap.ini, которые можно использовать для применения ограничения памяти к процессу soffice.bin в Windows. По умолчанию ограничение не повлияет на какие-либо подпроцессы, поскольку второй ключ добавляет JOB_OBJECT_LIMIT_SILENT_BREAKAWAY_OK в список флагов заданий. Чтобы активировать ограничение, расширьте bootstrap.ini следующим образом: [Win32] LimitMaximumMemoryInMB = 0 ExcludeChildProcessesFromLimit = true и измените значения в соответствии с вашими потребностями.
